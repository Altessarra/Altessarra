### 🏗️ Featured Project

**[BazaarX / PalengKART](#)** — Multi-sided marketplace platform
- Buyer, seller, rider, and admin roles in one ecosystem
- Web app (React/TypeScript + Vite), mobile app (React Native/Expo), and admin dashboard
- Supabase backend with RLS policies, real-time subscriptions, and PayMongo payment integration
- Features I've built: cart & checkout system, buyer settings + saved payment methods, voucher/discount engine, registry & gifting, seller earnings & payout reports, admin live-session monitoring, and more
